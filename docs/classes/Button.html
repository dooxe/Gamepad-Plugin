Button Class

Defined in: plugins/gamepad-plugin-1.0.0.js:279
Module: Gamepad
Parent Module: Kiwi.Plugins

The Button class is used to keep track of each button. Currently it only tracks index, value, and the pressed boolean. In future we are hoping to add functionality such as duration and more

  • [threshold] Number optional

    Between 0 - 1. This sets the threshold that a button must exceed before it is considered hit. Only works for triggers.
